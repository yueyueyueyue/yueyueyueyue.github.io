How to edit this site anywhere

1.download my private rsa key from somewhere like box.com

2.put the "id_rsa" file to the folder ".ssh"

3.install git kit. "git clone git@github.com:yueyueyueyue/yueyueyueyue.github.io.git"

4.do your edition.

5."git push origin master" all done!
