1.download my private rsa key from somewhere like box.com

2.put the "id_rsa" file into the folder ".ssh"

3.install git kit.

4.command: git clone git@github.com:yueyueyueyue/yueyueyueyue.github.io.git

5.command: git config --global user.email "ddgysx@yeah.net"

6.command: git config --global user.name "yueyueyueyue"

7.do your edition.

8.command: git push origin master

9. all done!

How to add a new project to github

Create a new repository in github. Then type those commands below in git bash.

1.command: git remote add origin git@github.com:yueyueyueyue/basic_cpp.git

2.command: git add -A

3.command: git commit -m "add all origin files"

4.command: git push -u origin master

5. all done!

Frenquently used steps in git bash

1.command: git pull

2.command: git add -A

3.command: git commit -m "update all changed files"

4.command: git push origin master

5. all done!
