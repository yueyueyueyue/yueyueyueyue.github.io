How to edit this site anywhere

1.download my private rsa key from somewhere like box.com

2.put the "id_rsa" file into the folder ".ssh"

3.install git kit.

4.command: git clone git@github.com:yueyueyueyue/yueyueyueyue.github.io.git

5.command: git config --global user.email "ddgysx@yeah.net"

6.command: git config --global user.name "yueyueyueyue"

7.do your edition.

8.command: git push origin master

9. all done!



By the way, my github account name is ddgysx@yeah.net. In case I forget it later.


use command "ssh-keygen -t rsa" to generate a rsa key pair.
