2.ZI = interp2(x, y, Z, XI, YI)
Interpolate ZI(XI, YI) using the point datas: (x, y, Z). The x and y is like 2:5.
I usually use this form: interp2(im, X, Y, 'linear', 0);
After using this, we can visulized it using this command:
surf(XI, YI, ZI);
