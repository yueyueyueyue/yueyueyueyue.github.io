                                                                                                                                                                                  Oleksandr Vasiliev Nico Christie a month ago

                                                                                                                                                                                  Hi, Nico!
                                                                                                                                                                                  Are you checking out your new branch on another computer? To see if you have really fetched the branch use *git branch*. If the branch is in the list, just *git checkout <branchname>* to it.

                                                                                                                                                                                  Edit: You have to create local branch that tracks changes on remote branch before all the above: *git checkout -b <branch-name> <remote>/<branch-name>*, where <remote> is typically *origin* (you can check your remote branches with *git branch -r*).

                                                                                                                                                                                  Hope this helps.
